# 🤝 Guia de Contribuição — Orae

Bem-vindo(a)!  
Agradecemos seu interesse em contribuir com o **Orae**, um aplicativo católico voltado à oração, espiritualidade e comunidade.  
Este documento explica como participar de forma organizada e profissional.

---

## 🧭 Regras Básicas

1. Sempre trabalhe em uma **branch separada** (nunca na `main` diretamente).  
2. Utilize nomes de branches padronizados:
3. Faça commits **pequenos e descritivos**.  
4. Teste localmente antes de enviar qualquer alteração.  
5. Mantenha a consistência com o design e o estilo do projeto.

---

## 💡 Tipos de Commits

| Tipo | Uso |
|------|-----|
| `feat` | Adição de nova funcionalidade |
| `fix` | Correção de erro |
| `docs` | Mudança na documentação |
| `style` | Ajuste visual ou de formatação |
| `refactor` | Melhorias no código existente |
| `test` | Adição ou ajuste de testes |

**Exemplo:**
```bash
git commit -m "feat: adiciona tela de pedidos de oração"
